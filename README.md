# E-commerce Customer Behavior Analysis

## 📌 Project Overview

This project analyzes customer behavior in an e-commerce platform. The analysis includes data preprocessing, exploratory data analysis (EDA), SQL queries, and conversion funnel analysis to identify user behavior patterns and business insights.

## 📂 Repository Structure

- `DataPreparation+ EAD.ipynb` — data cleaning and preprocessing.
- `Funnel.ipynb` — conversion funnel analysis.
- `SQL_analysis.ipynb` — SQL queries for business analysis.

## 🛠️ Tools & Technologies

- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- SQL

## 📈 Key Questions

- How do users interact with the platform?
- Where do users drop out of the conversion funnel?
- What insights can be obtained from SQL analysis?
- Which customer behaviors may impact conversion?

## 📊 Results

- Cleaned and prepared the dataset.
- Performed exploratory data analysis.
- Built a conversion funnel.
- Wrote SQL queries to answer business questions.
- Identified key user behavior patterns.

The dataset is publicly available on Kaggle.
